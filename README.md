# What to Wear WTWR

## Description
 WTWR is a full stack web application that helps users decide what to wear based on current weather conditions. Users can sign in, login, like items, dislike items, delete items, and manage their profile.

## Description of Technologies and techniques used

1. React FrontEnd
2. Express BackEnd
3. Mongo DB as a database
4. Moongoose
5. Jwt Authentication
6. Bcrypt for password hashing
7. Centerlized Error Handling
8. Winston Library for logging
9. PM2 for process management
10. Joi for Validation
11. Nginx reverse proxy
12. SSL via CERTBOT
13. Deployed on Goggole Cloud VM
